# CutFlow — 口播视频智能粗剪工具

CutFlow 是一个帮助视频剪辑师加速粗剪流程的工具。它能把编导脚本和录音音频自动对齐，识别出哪些片段该保留、哪些该删掉（比如口误、多余的停顿），然后生成可以直接导入 Premiere、Final Cut Pro、DaVinci Resolve、剪映等剪辑软件的时间线文件。

简单来说：**上传脚本和音频 → 自动处理 → 导出到剪辑软件 → 直接开始精剪。**

---

## 主要功能

- **智能对齐**：自动将录音内容与脚本逐句匹配，找到每句话在音频中的位置
- **口误识别**：自动标记说错的、重复录制的片段，你只需确认即可
- **停顿检测**：识别呼吸、思考间隔等，精确切分每个片段
- **交互审核**：在网页上逐段试听、查看匹配结果，随时手动调整
- **多格式导出**：支持 EDL（Premiere、DaVinci Resolve）、FCPXML（Final Cut Pro、剪映）、SRT 字幕
- **字幕词典**：自动修正转录中的常见错误，支持自定义纠错规则
- **批量处理**：可以同时处理多个任务，历史记录自动保存

## 两种转录方式

| | 本地转录 | 云端转录 |
|---|---|---|
| 是否需要联网 | 不需要 | 需要 |
| 速度 | 较慢 | 快 |
| 中文识别精度 | 好 | 更好 |
| 费用 | 免费 | 按用量付费 |

本地转录使用 Whisper 模型，在你自己的电脑上运行；云端转录使用火山引擎的语音识别服务，需要注册账号并获取密钥。

## 下载与安装

只需要打开 Mac 上的「终端」应用（在"启动台"搜索"终端"即可找到），依次粘贴以下命令并回车。

### 第一步：下载项目

```bash
cd ~/Desktop
git clone https://github.com/heyitseric/CutFlow.git
cd CutFlow
```

> **如果提示没有安装 git**：运行 `xcode-select --install`，按照弹窗提示安装后，重新运行上面的命令。

### 第二步：一键安装所有环境

```bash
chmod +x setup.sh
./setup.sh
```

这个脚本会自动检测并安装所有需要的软件（Homebrew、Python、Node.js、FFmpeg），以及 CutFlow 本身的依赖。已经装过的会自动跳过，不会重复安装。整个过程大约需要几分钟，中途可能会要求你输入电脑密码。

### 第三步：启动

```bash
./start.sh
```

启动后浏览器会自动打开 CutFlow 的界面。首次使用请在网页上方的「设置」页面中填写 AI 服务的密钥（支持火山方舟、OpenAI、DeepSeek 等）。

以后每次使用，只需要打开终端，输入：

```bash
cd ~/Desktop/CutFlow
./start.sh
```

## 配置云端转录（可选）

如果你想使用云端转录（更快、中文更准），需要在火山引擎注册账号并获取两组密钥。

### 1. 获取 LLM 智能匹配的 API Key（火山方舟）

- 打开 [火山方舟控制台](https://console.volcengine.com/ark)，注册 / 登录
- 创建 API Key，复制保存

### 2. 获取语音转录的 App ID 和 Token（豆包语音）

- 打开 [豆包语音控制台](https://console.volcengine.com/speech/app)，创建一个应用
- 记下 App ID 和 Token
- 官方文档参考：[音视频字幕生成](https://www.volcengine.com/docs/6561/80909?lang=zh)

### 3. 填写密钥

**推荐方式：** 启动 CutFlow 后，在网页上方点击「设置」，直接在页面中填写。

**手动方式：** 如果你更习惯直接编辑配置文件，可以修改项目根目录下的 `.env` 文件：

> **如何在 Mac 上找到并编辑 .env 文件：**
>
> `.env` 是一个隐藏文件（文件名以 `.` 开头的文件在 Mac 上默认不显示）。有两种方式打开它：
>
> **方式一：用终端打开（推荐）**
> ```bash
> open -a TextEdit ~/Desktop/CutFlow/.env
> ```
> 把这行命令粘贴到终端里回车，就会用「文本编辑」打开 .env 文件。编辑完后按 `⌘S` 保存即可。
>
> **方式二：在访达中显示隐藏文件**
> 1. 打开「访达」，进入 CutFlow 所在的文件夹（桌面上的 `CutFlow` 文件夹）
> 2. 按下键盘快捷键 `⌘ + Shift + .`（同时按 Command、Shift 和句号键）
> 3. 此时隐藏文件会变成半透明的样子显示出来，找到 `.env` 双击打开即可
> 4. 编辑完后再按一次 `⌘ + Shift + .` 可以重新隐藏这些文件

## 使用方法

1. 在网页界面点击「上传」，选择你的脚本文件（.md 或 .txt）和音频文件
2. 选择转录方式（本地或云端），点击开始处理
3. 处理完成后进入审核页面，逐段试听和确认
4. 确认无误后点击「导出」，选择你需要的格式
5. 把导出的文件导入你的剪辑软件

## 常见问题

**Q: 支持哪些音频格式？**
MP3、WAV、M4A、AAC 等常见格式都支持。

**Q: 支持英文吗？**
转录本身支持多语言，但匹配逻辑目前主要针对中文口播优化。

**Q: 导出的文件怎么导入剪辑软件？**
在剪辑软件中选择「导入」或「导入 XML」，选择导出的文件即可。

**Q: 启动后打不开网页怎么办？**
手动在浏览器地址栏输入 `http://localhost:5173` 访问。
